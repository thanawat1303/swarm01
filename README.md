## ขั้นตอนการติดตั้ง และใช้งาน ใน VM
 1. ทำ clone Template โดยใน Template ต้องมีการลง Docker และ set Date

 3. ทำการนำ Url Token ในการเชื่อมกับ Manage swarm จากเครื่อง Manage
